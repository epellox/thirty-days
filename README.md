# 30 Days. Then Gone.

An open source website with exactly 30 days to live. Anyone can fork it, add to it, build on it. Nothing gets deleted. Every contributor leaves their mark permanently — until April 17, 2026. Then it's gone.

No archive. No revival. No nostalgia.

---

## The Rules

1. **Add anything you want. Delete nothing.**
2. **Comment your code.** Strangers will read it.
3. **No malicious content.** You are responsible for what you submit.
4. **The end date is fixed.** The clock doesn't stop.
5. **On April 17, 2026 — it's gone.**

---

## How to Contribute

See [CONTRIBUTING.md](CONTRIBUTING.md) for the full guide.

The short version:
1. Fork this repo
2. Add your files to `/contributions/your-name/`
3. Sign the guestbook in `guestbook.json`
4. Open a PR to `main`

## Tech Stack

Plain HTML, CSS, and JavaScript. No frameworks. No build tools. No dependencies. Hosted on GitHub Pages. One repo, one branch.

This was an intentional choice — so that anyone can read, understand, and contribute to it without needing to learn a specific toolchain.
